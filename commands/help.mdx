## Overview

The `/help` command displays a formatted list of all available commands in OMNI, along with brief descriptions of what each command does.

## Usage

```bash
omni > /help
```

No arguments or flags are needed — just type the command and press Enter.

## Output

When you run `/help`, you'll see:

```bash
omni > /help

  Available commands:

  /help    — Show this help menu
  /clear   — Clear the screen
  /version — Show current version
  /about   — What is OMNI?
  /exit    — Quit the CLI
```

The help menu is formatted with:
- A cyan header ("Available commands:")
- Command names in white
- Descriptions in gray
- Clean spacing for easy reading

## When to use /help

Use the `/help` command when you:

- Forget which commands are available
- Want to quickly reference command names
- Need to remember what a specific command does
- Are new to OMNI and exploring its features

<Tip>
  The help menu stays visible on screen, so you can reference it while typing your next command.
</Tip>

## Implementation details

The `/help` command is implemented in the `handleCommand` function (src/app.js:22-24):

```javascript
case '/help':
  printHelp();
  return true;
```

The actual output is generated by the `printHelp` function in src/screen.js:40-50, which uses the `chalk` library to apply colors:

```javascript
function printHelp() {
  console.log('');
  console.log(chalk.cyan.bold('  Available commands:'));
  console.log('');
  console.log(chalk.white('  /help    ') + chalk.gray('— Show this help menu'));
  console.log(chalk.white('  /clear   ') + chalk.gray('— Clear the screen'));
  console.log(chalk.white('  /version ') + chalk.gray('— Show current version'));
  console.log(chalk.white('  /about   ') + chalk.gray('— What is OMNI?'));
  console.log(chalk.white('  /exit    ') + chalk.gray('— Quit the CLI'));
  console.log('');
}
```

<Note>
  After displaying help, the CLI returns to the prompt so you can immediately use any of the listed commands.
</Note>
